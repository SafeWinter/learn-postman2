## 1 模拟服务器的概念

基本概念——

- 本书中的模拟服务器（mock server），特指 API 接口的模拟服务器；
- 主要用途：模拟真实 API 服务器，可返回预设的 **硬编码** 响应。
- 既可以模拟单一端点（简单），也可以模拟多个端点（复杂），复杂程度取决于要测试的目标。
